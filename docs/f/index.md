# 公式

公式是 KWGT 的核心功能，它允许用户通过编程的方式来控制小部件的显示、属性以及其他逻辑。要详细了解公式，参见[百科 - 公式](/w/formula)。

## 入门

公式大多是相互连接的函数。公式的标识符是 `$`，它表示公式的开始和结束。公式中可以使用 `+` `-` `*` `/` `=` 等[数学运算符](./operator/)，也可以使用 `df` `bi` `tc` `tf` 等[函数](./function/)。

下面是一条简单的公式，它会显示类似 `2026/03/20 17:10:51` 的内容：

```kwgt
$df(yyyy/MM/dd HH:mm:ss)$
```

## 语法

### 函数

函数是公式的基本单元，它由函数名和参数组成。函数名是固定的，不能自定义函数。要查看所有可以使用的函数名，参见[函数](./function/)。

函数的基本结构是：

```kwgt
函数名(参数1, 参数2, ...)
```

### 嵌套

> 参见：[函数 - mu](./function/mu)

公式中的函数可以相互嵌套，形成复杂的逻辑。例如：

```kwgt
$mu(
    sqrt, mu(
        round, 2.80
    )
)$
```

你也可以写成单行形式：

```kwgt
$mu(sqrt, mu(round, 2.80))$
```

这会将 `2.80` 四舍五入到最接近的整数 (`3`)，然后再取平方根。
