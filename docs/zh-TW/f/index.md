# 公式

公式是 KWGT 的核心功能，它允許用戶通過編程的方式來控制小部件的顯示、屬性以及其他邏輯。要詳細瞭解公式，參見[百科 - 公式](/w/formula)。

## 入門

公式大多是相互連接的函數。公式的標識符是 `$`，它表示公式的開始和結束。公式中可以使用 `+` `-` `*` `/` `=` 等[數學運算符](./operator/)，也可以使用 `df` `bi` `tc` `tf` 等[函數](./function/)。

下面是一條簡單的公式，它會顯示類似 `2026/03/20 17:10:51` 的內容：

```kwgt
$df(yyyy/MM/dd HH:mm:ss)$
```

## 語法

### 函數

函數是公式的基本單元，它由函數名和參數組成。函數名是固定的，不能自定義函數。要查看所有可以使用的函數名，參見[函數](./function/)。

函數的基本結構是：

```kwgt
函數名(參數1, 參數2, ...)
```

### 嵌套

> 參見：[函數 - mu](./function/mu)

公式中的函數可以相互嵌套，形成複雜的邏輯。例如：

```kwgt
$mu(
    sqrt, mu(
        round, 2.80
    )
)$
```

你也可以寫成單行形式：

```kwgt
$mu(sqrt, mu(round, 2.80))$
```

這會將 `2.80` 四捨五入到最接近的整數 (`3`)，然後再取平方根。
